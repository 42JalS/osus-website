| API | HTTP 메서드 | 설명 |
| --- | --- | --- |
| url | POST | 단축 URL 생성하기|
| custom-url | POST | 원하는 단어로 단축 URL 생성하기|
| emoji-url | POST | 이모지로 단축 URL 생성하기|
| title-url | POST | 원본 문서의 타이틀 정보로 단축 URL 생성하기|
| original-url | GET | 단축 URL로 원본 URL 가져오기|
